Vamos aprender sobre Sofware?



Em sua terminologia Software é usado para retratar programas, scripts, firmware, a fim de ordenar o que os dispositivos devem fazer. Podemos definir Software como o conjunto de orientações que permite o usuário ter controle e autonomia sobre dispositivos, sejam eles desktop ou smartphones.



Fotos



Em exemplos software pode ser dito como uma smart TV, tablet, smartphone, um editor de texto, um app de streaming, como foi citado são inúmeras as formas de apresentação mas, podemos divididas em três categorias principais sendo elas software de programação, de sistema, de aplicação. Software de programação: essa categoria permite ao programador desenvolver novos software as quais usam linguagens diferentes para tal, sendo elas C, Java, Python, Swift, entre outras; Software de sistema: essa categoria é encarregada de fazer a comunicação entre o computador e o usuário, ou seja, serve como plataforma para o funcionamento de outros softwares; Software de aplicação: tem como função executar tarefas das mais diversas, tanto as de uso individual, quanto as de uso global.



Fotos



Hardware x Software Antes de apresentarmos as divergências entre Hardware e Software, é necessário saber o que é Hardware. Hardware é o usado como referência aos componentes físicos de um desktop, traduzindo, são os componentes que podem ser vistos e tocados, temos como exemplo a CPU, a placa-mãe, os dispositivos de entrada e saída. Partindo dessas definições podemos definir como hardware todos os componentes que podem ser vistos e apalpados em um desktop, como nos exemplos ditos acima, e Software é o conjunto de programas e orientações que fazem com que o hardware funcione.
